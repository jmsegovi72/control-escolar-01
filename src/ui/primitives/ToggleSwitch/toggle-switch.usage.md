# ToggleSwitch

Usa `ToggleSwitch` para cambios de estado inmediatos como activar/desactivar una cuenta, autocolapso del sidebar o habilitar recordatorios.

## Cuando usarlo

- Estado binario con efecto inmediato.
- Configuraciones de producto.
- Preferencias que se pueden encender o apagar.

## Cuando no usarlo

- Seleccion de filas o items.
- Consentimientos tipo "acepto".
- Casos donde varias opciones compiten entre si.

## Props clave

- `checked`
- `disabled`
- `invalid`
- `description`
- `showState`
- `onLabel`
- `offLabel`
- `size`
